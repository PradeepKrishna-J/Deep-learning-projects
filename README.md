# üß† Deep Learning Projects

A collection of hands-on deep learning implementations covering fundamental concepts to advanced neural network architectures. Each project is implemented in Jupyter notebooks with detailed code, explanations, and visualizations.

## üìö Projects

### 1. **Linear Regression** 
Implementation of linear regression from scratch using gradient descent optimization.
- Custom gradient descent algorithm
- Loss curve visualization
- Model evaluation on synthetic data

### 2. **Softmax Regression**
Multi-class classification on the Iris dataset using softmax regression.
- One-hot encoding
- Cross-entropy loss
- 100% test accuracy achieved

### 3. **XOR using MLP**
Solving the XOR problem with a Multi-Layer Perceptron to demonstrate neural networks can learn non-linear functions.
- 2-layer neural network
- Backpropagation from scratch
- Sigmoid activation

### 4. **MNIST CNN**
Convolutional Neural Network for handwritten digit recognition.
- Deep CNN architecture with batch normalization
- 99.07% test accuracy
- Dropout regularization

### 5. **IMDB Sentiment Classification using LSTM**
Sentiment analysis on movie reviews using LSTM networks.
- Text preprocessing and tokenization
- Stacked LSTM layers
- Binary sentiment classification

### 6. **CIFAR-10 Image Classification with InceptionV3**
Transfer learning approach using pre-trained InceptionV3 for image classification.
- Transfer learning with frozen layers
- 62% accuracy on CIFAR-10
- Confusion matrix visualization

### 7. **MultiHead Attention with PyTorch TransformerEncoder**
Implementation of transformer architecture and multi-head attention mechanism.
- Self-attention mechanism
- Positional encoding
- PyTorch implementation

## üöÄ Getting Started

### Prerequisites
```bash
Python 3.8+
Jupyter Notebook
```

### Installation

1. Clone the repository:
```bash
git clone https://github.com/PradeepKrishna-J/Deep-learning-projects.git
cd Deep-learning-projects
```

2. Install required packages:
```bash
pip install numpy pandas matplotlib seaborn
pip install tensorflow keras torch torchvision
pip install scikit-learn jupyter
```

3. Launch Jupyter Notebook:
```bash
jupyter notebook
```
